Ho utilizzato la piattaforma Github per le varie fasi di programmazione e per mantenere sincronizzati 3 computer su cui abitualmente lavoro.

I comandi principali che ho utilizzato sono stati i seguenti:

git push -u origin master
per caricare la cartella di lavoro locale su server

git pull origin master
per scaricare la cartella di lavoro su un un altro computer

La lista completa di comandi che ho utilizzato sono stati i seguenti:

git init
// per inizializzare una cartella locale

git remote add origin git@github.com:francescocarraro77/ProgettoIng2.git
// per impostare la repository server

git add .
// per aggiungere la repository nella staging area

git commit -m "first commit"
// per committare il lavoro

git push -u origin master
// per caricare la repository su server

git pull origin master
// per scaricare la repository in locale

git log --oneline
// per visualizzare la lista dei commit

git checkout master
// per tornare alla testa dei commit

ssh-keygen -t rsa -C "francesco.carraro@studenti.unitn.it"
// verrà creata una nuova chiave in c:\users\XXXXX\.ssh\id_rsa.pub che bisogna inserire nelle impostazioni web di github

npm test
// per lanciare jest che andrà a cercare in tutta la repository eventuali file *.test.js

Heroku

Install Heroku CLI
https://devcenter.heroku.com/articles/heroku-cli
CLI log in:
heroku login

Comandi Win64OpenSSL_Light-1_1_1m per generare le chiavi da scaricare in locale:

openssl genrsa -out rsa.key 1024
// per generare chiave privata

openssl rsa -in rsa.key -out rsa.pem -pubout -outform PEM
// per generare la chiave pubblica
